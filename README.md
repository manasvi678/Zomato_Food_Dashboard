# Zomato_Food_Dashboard
🎯 **Project Objective**  
To design and develop an interactive and visually engaging Food Delivery Dashboard using Python and Power BI that provides a comprehensive overview of Zomato restaurant data. The dashboard enables insights into orders, delivery performance, customer ratings, and trends across cities, order types, and delivery metrics for data-driven decision-making.

---

✅ **Features**

📅 **Monthly/City-wise Orders Overview**  
Track the number of orders by city and month to identify trends and high-performing locations.

📦 **Order Type & Restaurant Analysis**  
Analyze how different types of orders (delivery, takeaway) and restaurants contributed to overall performance.

🌆 **Geographic Insights**  
Understand delivery performance across different cities and regions.

⏱️ **Delivery Time & Traffic Analysis**  
Visualize the impact of traffic and weather on delivery times.

⭐ **Customer Ratings Overview**  
Monitor average restaurant ratings to assess customer satisfaction.

📊 **Interactive Power BI Charts**  
Bar charts, pie charts, and KPI cards provide a clean and professional visual summary.

🧹 **Clean Layout with Auto-updating Visuals**  
Dashboard automatically updates when new data is added to the cleaned dataset.

---

🛠️ **Tools Used**  
- Python (pandas, matplotlib, seaborn) – for data cleaning and EDA  
- Power BI Desktop – for dashboard creation  
- CSV Dataset (clean_zomato_dataset.csv) – cleaned data ready for analysis  
- Microsoft Excel (optional) – for quick data inspection

🚀 **How to Use**

1. Open `analysis.ipynb` in Jupyter Notebook or VS Code to explore data cleaning and EDA.  
2. Open `dashboard/Zomato_Food_Delivery_Dashboard.pbix` in Power BI Desktop to explore the interactive dashboard.  
3. To update visuals, replace or append new data in `clean_zomato_dataset.csv`; Power BI visuals will refresh automatically.  
4. Use filters or slicers in Power BI to analyze specific cities, order types, or restaurants.
